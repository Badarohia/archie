# Archie

> A modular, local-first AI platform.

## Vision

Archie is a long-term AI platform designed to evolve from a personal assistant into a fully capable autonomous AI system while maintaining a stable, scalable architecture.

## Goals

- Local-first
- Modular architecture
- Long-term memory
- Voice interaction
- Desktop automation
- Future live-streaming support
- Hardware-independent design

## Status

🚧 Under active development.

## Repository Structure

- `src/` — Application source code
- `config/` — Configuration files
- `docs/` — Documentation
- `storage/` — Runtime data
- `tests/` — Test suite
